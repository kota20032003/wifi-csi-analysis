# Wi-Fi CSI Analysis

## 概要

卒業研究で使用したWi-Fi CSI解析用のPythonコードです．
ESP32から取得したCSIデータを用いて，CSIの取得，3D可視化，機械学習による分類，予測結果の後処理を行います．

## 背景

Wi-Fi CSI (Channel State Information: チャネル状態情報) は，Wi-Fi信号が空間を伝わるときの変化を表す情報です．
本研究では，ESP32で取得したCSIデータを用いて，環境変化や状態変化を観測することを目的としました．

## 主な機能

- ESP32からCSIデータを取得し，CSVに保存
- CSIの振幅をリアルタイムに可視化
- 時間，周波数，振幅の3Dグラフを表示
- IQ特徴量と振幅特徴量を用いたロジスティック回帰
- 状態遷移ルールによる予測ラベルのスムージング
- 多数決による予測ラベルのスムージング

## 各ファイルの説明

| ファイル名 | 役割 |
|---|---|
| `plot_csi_ravel2_old.py` | ESP32からCSIデータを取得し，CSVに保存するコードです．指定したサブキャリアのI/Q成分から振幅を計算し，リアルタイムにグラフ表示します． |
| `plot_3d_freq.py` | 取得したCSIデータを，時間，周波数，振幅の3軸で3D表示するコードです．サブキャリアごとの振幅変化を視覚的に確認できます． |
| `csi_experiment.py` | CSIデータを用いて，ロジスティック回帰による分類実験を行うコードです．IQ特徴量と振幅特徴量を作成し，複数条件で分類性能を比較します． |
| `smooth_fsm.py` | 予測ラベルに対して，状態遷移ルールに基づく補正を行うコードです．不自然なラベル変化を抑えるために使用します． |
| `smooth_majority.py` | 予測ラベルに対して，多数決によるスムージングを行うコードです．瞬間的な誤分類や細かい揺れを抑えるために使用します． |

## 主な関数の説明

### `plot_csi_ravel2_old.py`

| 関数名 | 説明 |
|---|---|
| `ts_iso_us_slash()` | 現在時刻をマイクロ秒単位の文字列に変換する関数です．CSIデータをCSVに保存するとき，取得時刻を細かく記録するために使用します． |
| `csi_data_handler()` | シリアルポートからCSIデータを読み取り，CSIペイロードの抽出，CSV保存，グラフ表示用データの更新を行う関数です． |

### `plot_3d_freq.py`

| 関数名 | 説明 |
|---|---|
| `build_freq_axis_mhz()` | 通信帯域幅とFFTサイズから，サブキャリアに対応する周波数軸を作成する関数です． |
| `payload_to_amplitudes()` | `csi_payload` の文字列を数値列に変換し，I/Q成分から振幅を計算する関数です． |
| `detect_wall_bin_from_train()` | `train.csv` を読み込み，各サブキャリアの平均振幅から突出して大きいビンを検出する関数です． |
| `csi_data_handler()` | シリアル通信でCSIデータを取得し，I/Q成分から振幅を計算して，3D表示用の履歴データに追加する関数です． |
| `setup_big_window()` | Matplotlibの表示ウィンドウをできるだけ大きく表示するための関数です． |
| `main()` | CSI取得スレッドを開始し，3Dグラフを更新し続けるメイン処理です． |

### `csi_experiment.py`

| 関数名 | 説明 |
|---|---|
| `parse_iq_pairs()` | `csi_payload` の文字列を，I/Q成分のペアに変換する関数です． |
| `make_features_iq()` | I/Q成分をそのまま特徴量として使用するための関数です．ガード帯やDC成分を除外し，1次元の特徴量ベクトルに変換します． |
| `make_features_amp()` | I/Q成分から振幅を計算し，振幅特徴量を作成する関数です．振幅は `sqrt(I^2 + Q^2)` で計算します． |
| `run_experiment()` | ロジスティック回帰の学習，予測，評価，CSV保存をまとめて行う関数です．正解率，適合率，再現率，F値，混同行列を出力します． |
| `main()` | 学習データとテストデータを読み込み，4パターンの分類実験を実行するメイン処理です． |

### `smooth_fsm.py`

| 関数名 | 説明 |
|---|---|
| `next_allowed_states()` | 現在の状態から次に遷移してよい状態の集合を返す関数です． |
| `smooth_with_fsm()` | 状態遷移ルールに基づいて予測ラベルを補正する関数です．許可されていない遷移が出た場合は，前の状態を維持します． |
| `main()` | 予測結果CSVを読み込み，状態遷移ルールによるスムージングを行い，補正後のCSVを保存するメイン処理です． |

### `smooth_majority.py`

| 関数名 | 説明 |
|---|---|
| `smooth_with_majority()` | 指定した窓幅の中で多数決を行い，予測ラベルをなめらかにする関数です．同数の場合は元のラベルを優先します． |
| `main()` | 予測結果CSVを読み込み，多数決スムージングを行い，補正後のCSVを保存するメイン処理です． |

## 工夫した点

| 工夫した点 | 内容 |
|---|---|
| CSIデータを扱いやすい形に変換 | ESP32から取得したCSIデータは，I/Q成分が文字列として出力されます．そのままでは解析しにくいため，I/Qペアへの変換，振幅計算，特徴量ベクトル化を行いました． |
| ガード帯とDC成分の除外 | データを載せないサブキャリアやDC成分を除外し，分類に使う特徴量を整理しました． |
| 複数特徴量での比較 | I/Q特徴量と振幅特徴量の両方を用意し，どちらが分類に有効か比較できるようにしました． |
| クラス不均衡への対応 | クラス重みなしとクラス重みありの両方で実験し，データ数の偏りが分類結果に与える影響を確認できるようにしました． |
| 複数の評価指標を出力 | 正解率だけでなく，適合率，再現率，F値，混同行列を出力し，分類結果を多面的に確認できるようにしました． |
| リアルタイム可視化 | CSIを取得しながら振幅変化を表示できるようにし，実験中に信号の変化を確認できるようにしました． |
| 予測結果の後処理 | 状態遷移ルールと多数決スムージングを実装し，瞬間的な誤分類や不自然なラベル変化を抑えるようにしました． |
| 設定値を上部に集約 | COMポート番号，ボーレート，窓幅，出力ファイル名などをコード上部にまとめ，実験環境に合わせて変更しやすくしました． |
| 異常な入力への対応 | CSIデータの形式が崩れている場合やデータ数が不足している場合でも，処理が止まりにくいようにしました． |
